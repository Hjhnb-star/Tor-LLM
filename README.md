
本项目是面向Tor暗网出口明文流量的智能威胁检测系统，依托大语言模型(LLM)的深度语义理解与上下文推理能力，结合自研CWE-CVE本地化威胁情报知识图谱，实现从原始pcap流量解析、多维度特征提取到已知/未知威胁精准识别、漏洞匹配与攻击链还原的全流程自动化分析，突破传统检测方案对预定义特征库的强依赖，完成从「被动特征匹配」到「主动行为认知」的技术跃迁，实现对0day攻击的早期感知。

---

## 📌 项目背景
### 行业痛点
Tor洋葱网络通过多层加密与分布式节点转发实现匿名通信，在提供隐私保护的同时，也成为网络勒索、数据泄露、高端目标渗透等违法犯罪活动的温床：
- 2025年4月，透析巨头DaVita遭勒索攻击，91.5万患者敏感信息在暗网泄露1.51TB数据
- 2025年8月，黑客组织通过Tor暗网泄露蒙特卡洛赌场运营方SBM集团核心数据，严重威胁其全球声誉

传统流量检测方案面临无法突破的瓶颈：
1. 基于规则匹配与统计特征的方案，严重依赖标注数据集，对新型对抗样本与0day攻击束手无策
2. 深度学习方案可解释性低、误报率高，无法输出完整攻击链逻辑与溯源依据
3. 针对IoT设备的暗网定向攻击检测精度不足，漏洞匹配误报率高

### 合规要求
本项目贴合《网络安全法》《数据安全法》《个人信息保护法》形成的网络空间治理法律框架，满足2025年1月1日施行的《网络数据安全管理条例》对网络威胁监测与数据安全防护的核心要求。

## 🎯 项目概述
本项目核心架构采用**「分层解析-特征融合-智能决策」三阶处理机制**，依托LLM强大的语义解析能力，结合本地化构建的CWE-CVE威胁情报知识图谱，实现从原始流量特征提取到深度威胁识别的全流程覆盖。系统针对Tor出口路由器解密后的明文流量进行深度分析，可精准识别主流网络攻击行为，并具备对未知威胁的迁移性识别能力，为暗网空间治理提供智能化技术支撑。

## 🏗️ 核心架构与技术流程
### 一阶：分层解析层 —— 多维度流量特征深度提取
完成原始流量的标准化处理与全维度特征解析，为后续分析提供结构化数据基础：
1. **流量标准化预处理**：解析原始pcap文件，基于五元组完成会话聚合，生成结构化流数据，累计处理12个月42.3TB Tor明文流量
2. **统计特征解析**：提取时频特征、包大小分布、请求响应比、时间间隔方差等量化指标，识别异常通信行为
3. **语义特征解析**：对HTTP请求/响应全量内容进行深层语义解析，梳理业务交互逻辑，识别语义冲突与异常载荷
4. **IoT专项解析**：定向提取设备型号、固件版本、特定访问路径等标识信息，实现IoT设备攻击的靶向检测

### 二阶：特征融合层 —— LLM语义增强与知识图谱联动
实现量化特征与语义特征的融合，完成攻击行为的结构化描述与威胁情报关联：
1. **LLM语义增强**：基于提示词工程与COT思维链技术，引导LLM将离散流量特征转化为标准化攻击类型与手法描述，输出可解释的推理过程
2. **威胁情报知识图谱**：基于Qwen3-Embedding-4B量化模型，本地化构建CWE-CVE知识图谱，覆盖1000+攻击类型与漏洞的关联映射关系
3. **两阶段RAG检索**：先基于攻击语义描述匹配对应CWE弱点类型，再在关联CVE范围内进行语义精搜，实现漏洞的精准匹配，大幅降低误报率

### 三阶：智能决策层 —— 威胁综合研判与未知威胁识别
完成多维度信息的融合研判，输出最终检测结果与处置建议：
1. **已知威胁精准匹配**：对特征与CVE描述高度吻合的流量，直接输出标准化威胁分析报告，包含威胁等级、攻击意图、漏洞编号与置信度
2. **未知威胁迁移识别**：对CVE匹配失败或匹配度低的流量，启动未知攻击识别模式，基于语义异常与行为模式推理，识别新型攻击逻辑，输出人工复核建议
3. **全流程自动化输出**：端到端完成从流量输入到威胁报告生成的全流程处理，支持批量流量的无人值守分析

## ✨ 核心功能与技术亮点
1. **无特征库依赖的未知威胁识别**：基于LLM深层语义理解能力，无需预定义规则，即可从流量文本的逻辑冲突、语义异常中发现新型攻击，突破传统方案对已知特征库的强依赖
2. **IoT设备靶向检测能力**：针对IoT设备流量设计专属分析逻辑，建立「设备特征-漏洞」的确定性映射机制，解决传统方案IoT漏洞匹配高误报的行业痛点
3. **高可解释性检测输出**：基于COT思维链优化，输出完整的推理过程与判断依据，而非单纯的黑白标签，满足安全运营的溯源与取证需求
4. **优异的检测性能**：在400条Tor流量测试样本中，实现**召回率91.72%、准确率90.75%、精确率95.11%、F1分数93.38%**的综合性能，在漏报与误报之间实现优秀平衡
5. **全流程自动化部署**：完成从流量预处理到报告输出的端到端自动化Pipeline开发，支持批量流量的并行处理，大幅提升暗网流量分析效率
